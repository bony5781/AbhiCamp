# AbhiCamp

A full-stack campground discovery and review web application built using Node.js, Express.js, MongoDB, and EJS.

## Features

- User Authentication & Authorization
- Create, Edit & Delete Campgrounds
- Campground Reviews & Ratings
- Cloudinary Image Uploads
- Image Carousel Gallery
- Interactive Maps with Leaflet
- Location Search with OpenStreetMap Geocoder
- Secure Route Protection
- Responsive UI using Bootstrap
- Flash Messages & Form Validation

## Tech Stack

- Node.js
- Express.js
- MongoDB
- EJS
- Bootstrap
- Cloudinary
- Leaflet.js
- Passport.js
